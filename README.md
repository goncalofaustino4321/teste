# @goncas_pro650 - Website Oficial

Bem-vindo ao website oficial do canal YouTube @goncas_pro650!

## 📋 Sobre o Projeto

Este é um site profissional e responsivo dedicado ao canal YouTube @goncas_pro650. O projeto apresenta:

- **Design Moderno**: Interface limpa e profissional
- **Responsivo**: Funciona perfeitamente em desktop, tablet e mobile
- **Performance**: Carregamento rápido e otimizado
- **SEO Friendly**: Estrutura HTML semântica
- **Interativo**: Animações suaves e transições elegantes

- Informações de contato

### 7. Footer
- Links sociais

## 🚀 Como Usar

### Instalação Local

1. Clone o repositório:
```bash
git clone https://github.com/goncalofaustino4321/teste.git
cd teste
```

2. Abra o arquivo `index.html` no seu navegador:
```bash
open index.html
# ou
start index.html
```

### Deploy Online

Você pode fazer o deploy deste projeto usando:

- **GitHub Pages**: Ative no Settings do repositório
- **Vercel**: Importe o repositório
- **Netlify**: Conecte o repositório
- **Qualquer hosting web**: Faça upload dos arquivos

## 🔧 Customização

### Alterar Cores
Edite as variáveis CSS no arquivo `styles.css`:

```css
:root {
    --primary-color: #ff0000;
    --secondary-color: #ffffff;
    /* ... */
}
```

### Adicionar Novos Vídeos
Duplique um `.video-card` na seção de vídeos e atualize as informações:

```html
<div class="video-card">
    <div class="video-thumbnail">
        <i class="fab fa-youtube"></i>
        <span class="duration">XX:XX</span>
    </div>
    <div class="video-info">
        <h4>Título do Vídeo</h4>
        <p class="views">X visualizações • há X dias</p>
        <p class="description">Descrição...</p>
        <a href="#" class="watch-btn">Assistir</a>
    </div>
</div>
```

### Atualizar Redes Sociais
Atualize os links no footer e na seção de contato:

```html
<a href="https://seu-link" target="_blank">Link Social</a>
```
